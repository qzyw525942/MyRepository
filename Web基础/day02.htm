第一集   javaScript概述

 

html是指静态页面，css是美化html的，javascript是让html动起来。html就好像是变形金刚的素描，css就是给变形金刚上色，javascript就是让变形金刚动起来。
 

javaScript由三部分组成：

  • 核心（ECMAScript）
  • 文档对象模型（DOM） Document object model
  • 浏览器对象模型（BOM） Broswer object model

 

html和js的两种结合方式：

方式1：

<!DOCTYPE html>

<html>

  <head>

    <title>TEST1.html</title>

        

    <meta http-equiv="keywords" content="keyword1,keyword2,keyword3">

    <meta http-equiv="description" content="this is my page">

    <meta http-equiv="content-type" content="text/html; charset=UTF-8">

   

<!--<link rel="stylesheet" type="text/css" href="./styles.css">-->

 

                   <!-- 使用script标签. 在标签中书写js代码 -->

                   <script type="text/javascript">

                            alert("hello world!");//alert相当于system.out.println

                   </script>

  </head>

 

  <body>

  </body>

</html>

第三集  BOM

 

BOM=>browser object model=>浏览器对象模型

 

将浏览器的各个部分封装成对象，我们可以使用这些对象来操作浏览器的一些动作

 

Bom的组成：5个对象

    Window：窗口对象，重点掌握

Location：把浏览器的地址栏封装成对象

History：把浏览器历史记录封装成对象

 

Screen：没用，把屏幕信息封装成对象

Navigation：没用，将浏览器封装成对象

 

只学前三个。

一个窗口就是一个window对象
