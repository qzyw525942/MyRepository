第一集  HTML

 

HTML语言是用于展示网页内容的语言。以下使用的都是MyEclipse工具。<!--  -->的快捷键是：ctrl+shift+/。格式化代码的快捷键是ctrl+shift+f。

 

在MyEclipse中新建java工程-->新建Folder-->新建File--->以html作为后缀即得到html文件。

 

案例一：

<html><!-- html的根标签,代表html文档的开始和结束 -->

         <head><!-- html的头部分 , 用以设置网页属性和标题 -->

                   <title>hello world</title><!-- 标题 -->

         </head>

         <body><!-- html的正文部分,放置想要在页面上显示的内容 -->

                   hello world!

         </body>

</html>

案例十一：(重点掌握)

要实现的效果如下：

<html>

<head>

<title>表格标记</title>

<meta http-equiv="Content-Type" content="text/html;charset=utf-8">

</head>

<body>

         <!-- form标签：用于说明表单的范围

                            action属性: 决定要把数据提交到哪儿去，#表示提交给表单自己

 

                   input标签

                            type属性:决定输入的类型

                            name属性(重点):提交的键名，每一个input都要写上

                            size属性:限制文本输入框的显示长度，适于input为text和password时

                            maxlength属性:限制文本输入框的输入长度，适于input为text和password时

                            readonly属性:只读，不可修改文本内容，但是数据会被提交，适于input为text                          和password时

                            disabled属性:禁用.被禁用的表单项，数据不会被提交. 所有input都可以使用

                           

                   select标签

                            multiple属性: 多选

                            size属性:一次可以选择的选项数量

                            disabled属性:禁用

 

                   textarea标签

                            disabled属性:禁用

                            readonly属性:只读

 

                   form

                            action:决定要把数据提交到哪儿去，#表示提交给表单自己

                            method:表单提交的方式.

                                     get提交:

                                               1.将参数键值对拼装在Url地址之后.

                                               2.get提交安全性相对较差

                                               3.get提交参数长度有限

                                      post提交:

                                               1.参数不在Url上

                                               2.post提交安全性相对较好

                                               3.post提交理论上参数长度没有限制

         -->

         <form action="#" method="post" >

 

                   用户名:<input type="text" name="username" ><br/>

<!--表单提交的数据都是以键值对的形式进行的，name属性即是键，输入的文本内          容即是值，下面的password亦同-->

 

                   密码:<input type="password" name="password" /><br/>

 

                   性别:男<input type="radio" name="sex" value="male" />

女<input type="radio" name="sex" value="female" /><br/>

                   <!-- 使用同样的name表示二者是一组，每次只能选择一个。自己提供键值-->

 

                   爱好:抽烟<input type="checkbox" name="habbit" value="smork" />

                            烫头<input type="checkbox" name="habbit" value="tangtou" /><br/>

                            <!-- 使用同样的name表示它们是一组。自己提供键值 -->

 

                   学历:<select name="edu" multiple size="10" >

                                     <option value="zk" >专科</option>

                                     <option value="bk" >本科</option>

                                     <option value="ss" >硕士</option>

                            </select><br/>

<!--键为name，值为三个value-->

 

                   个人说明:<textarea rows="10" cols="35" name="desc"  >这家伙很懒!神马也没留                下!</textarea><br/>

<!--键为name，值为输入的文本-->

<!--键为name，值为输入的数据-->

 

                   <input type="hidden" name="haha" value="heihei" />

<!-- 网页上不显示，但是数据却能提交 -->

         </form>

案例8：标签状态的改变

<!DOCTYPE html>

<html>

    <title>TEST8.html</title>

    <meta http-equiv="description" content="this is my page">

    <meta http-equiv="content-type" content="text/html; charset=UTF-8">

   

<!--

         伪类选择器:

                   选择标签的某个状态.需要配合其他选择器来使用

                   l link 未访问过

                   v visited   访问过

                   h hover     悬浮

                   a active    激活,点击

        

 -->

        <style type="text/css">

                 a:link{  /* 未访问a标签时呈绿色 */

                           color:green;

                 }

                 a:visited{  /* 已访问a标签时呈黑色 */

                           color:black;

                 }

                  

                 a:hover{  /* 鼠标在a标签上时呈白色 */

                           color:white;

                 }

                 a:active{  /*点击a标签时呈粉色 */

                           color:pink;

                 }

        </style>

  </head>

 

  <body>

        <a href="TEST1.html" >点我</a>

  </body>

</html>

案例13：盒子模型

<!DOCTYPE html>

<html>

  <head>

    <title>TEST13.html</title>

        

    <meta http-equiv="keywords" content="keyword1,keyword2,keyword3">

    <meta http-equiv="description" content="this is my page">

    <meta http-equiv="content-type" content="text/html; charset=UTF-8">

   

   

    <style type="text/css">

    /*

             盒子模型的属性

             一. 边框属性

             二. 尺寸属性

             三. 边距属性

                                *内边距

                                *外边距

   

   

             border-color:边框颜色

             border-width:边框宽度

             border-style:边框样式

            

             border-color: red;

             border-width: 1px;

             border-style: solid;

                      

             margin-left:100px;左外边距

             margin-top:100px;

                      

             padding-left:100px; 左内边距

             padding-top:100px; 上内边距

                      

             注意:设置内边距会改变所在外div的长或宽。

             内边距和外边距是相对而言的。内div的外边距就是外div的内边距

   

            

             #one{  /* 设置外div的属性 */

                       height: 300px;

                       width: 300px;

             }

             #two{  /* 设置内div的属性 */

                       height: 100px;

                       width: 100px;

                       margin-left:100px;

                       margin-top:100px;

             }

    </style>

   

  </head>

 

  <body>

       <div id="one" >

                <div id="two"></div>

       </div>

  </body>

</html>
