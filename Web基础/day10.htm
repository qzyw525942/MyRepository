A.page指令:

描述页面的一些属性

 

新建一个web工程自带一个index.jsp文件，它的第一行就是page指令：

<%@ page

language="java"    //说明JSP的嵌入式语言支持java语言

import="java.util.*,java.io.*"  //java语言所导入的包

pageEncoding="UTF-8"   //页面编码。在MyEclipse首选项中搜索jsp，如下设置就可以使以                                          //后新建的jsp都为UTF-8:

       

  //此外还有以下几个属性

autoFlush="true" buffer="8kb"  //服务器把jsp发送到浏览器时，会先经过缓冲区。前一个是                                                   //自动刷新，后一个是缓冲区的大小

errorPage="/page/error.jsp"  //如果index.jsp出现异常，就会转到error.jsp

session="true"                                     //当前页面是否可以直接使用session对象

 %>

(1)、out对象：

<body>

       response.getWriter().write("b");

Jsp向外输出流时，可以使用out流和response流，但是后者的优先级较高，所以其会优先输出，所以上面代码的最终结果将是bd ac。为了避免优先级的混乱，JSP中统一使用out流。

第三集    JSP中的标签

 

1、jsp:forward标签

<body>

 

   <%-- JSP动作标签。作用：分担jsp页面的java代码--%>

   

    <jsp:forward page="/index.jsp"></jsp:forward>

    <%--相当于如下请求转发代码 --%>

        <%--

             request.getRequestDispatcher("/index.jsp").forward(request, response);

     --%>

  </body>

 

这样，当访问本jsp时，就会转而访问index.jsp。

 

2、jsp:include标签

 <body>

    

<jsp:include page="/index.jsp"></jsp:include>

        <%--相当于如下请求包含代码 --%>

    <%--

         request.getRequestDispatcher("/index.jsp").include(request, response);

    --%>

  </body>

 

这样，当访问本jsp时，就会把本jsp和index.jsp的内容一起呈现出来。

MyDateConverter类：

public class MyDateConverter implements Converter

{

         // 自己填写转换代码

         // 参数1: 没用, 告诉实现者,需要转换成什么类型

         // 参数2: 待转换的jsp表单提交的参数

         // 返回值: 转换结果

         public Object convert(Class arg0, Object arg1)

         {

                   // 1 创建sdf

                   SimpleDateFormat sdf = new SimpleDateFormat("yyyy/MM/dd");

                   // 表单提交的日期类型的格式

 

                   // 2 转换参数2

                   String dateStr = arg1.toString();

                   // 表单提交的参数是String类型的，所以先把方法传递进来的参数arg1转换为                 //String类型

                   try

                   {

                            Date date = sdf.parse(dateStr);

 

                            // 3 将转换结果返回

                            return date;

                   } catch (ParseException e)

                   {

                            e.printStackTrace();

                            return null;

                   }

 

         }

 

}

//javaBean

//1.要求为属性提供get/set方法任意之一

public class User

{

         private String name;

         private String password;

         private int age;

         private Date hiredate;

 

         public Date getHiredate()

         {

                   return hiredate;

         }

 

         public void setHiredate(Date hiredate)

         {

                   this.hiredate = hiredate;

         }

 

         public int getAge()

         {

                   return age;

         }

 

         public void setAge(int age)

         {

                   this.age = age;

         }

 

         public String getName()

         {

                   return name;

         }

 

         public void setName(String name)

         {

                   this.name = name;

         }

 

         public String getPassword()

         {

                   return password;

         }

 

         public void setPassword(String password)

         {

                   this.password = password;

         }

 

         @Override

         public String toString()

         {

                   return "User [name=" + name + ", password=" + password + ", age=" + age

                                     + ", hiredate=" + hiredate + "]";

         }

 

}
