1:正则表达式（在类Pattern中）

       (1)就是规定了特定规则的字符串

       (2)常见规则

A:字符

x      字符 x。即任意一个字符表示该字符本身，如'a'表示字符a

\\     反斜线字符。 两条反斜线表示一条反斜线，是转义字符的表达形式

\n     换行符

\r     回车符

 

B:字符类

[abc]         a、b 或 c（简单类）只能出现一个，不能同时出现几个

[^abc]        任何字符，除了 a、b 或 c（否定）

[a-zA-Z]      a到 z 或 A到 Z，两头的字母包括在内（范围），即包括了所有的英文字母

[0-9]         0到9的字符都包括（[2-9]表示从2到9）

      

C:预定义字符类

.     任何字符。如果本身就是.字符本身，怎么表示呢? \.

\d    数字：[0-9]，所以[1-9][0-9]{4,14}等价于[1-9]\\d{4,14}，d前面的\线用两条表示一条

\w    单词字符：[a-zA-Z_0-9]，在正则表达式里面组成单词的东西必须由这些东西组成

 

D:边界匹配器

^    行的开头

$    行的结尾

\b   单词边界，就是不是单词字符的地方。就是把单词隔开的地方，如下面的空格，？和；

              举例：hello world?haha;xixi

      

E:数量词

X?        X出现一次或一次也没有（0或1）

X*        X出现零次或多次（0或1或多）

X+        X出现一次或多次（1或多）

 

X{n}       X出现恰好n次（n）

X{n,}       X出现至少n次 (>=n)

X{n,m}     X出现至少n次，但是不超过 m 次(>=n并且<=m)

 

所以：[1-9]\\d{4,14}的意思就是：最开始一位为1~9出现一次，后面的几位0~9出现至少4次，但不超过14次，所以总的为5~15位

/*

 * 请写代码实现最终输出结果是："27 38 46 50 91"

 *

 * 分析：

 *        A:定义一个字符串

 *        B:把字符串进行分割，得到一个字符串数组

 *        C:把字符串数组变换成int数组

 *        D:对int数组排序

 *        E:把排序后的int数组在组装成一个字符串

 *        F:输出字符串

 */

public class RegexTest {

       public static void main(String[] args) {

              // 定义一个字符串

              String s = "91 27 46 38 50";

 

              // 把字符串进行分割，得到一个字符串数组

              String[] strArray = s.split(" ");

 

              // 把字符串数组变换成int数组

              int[] arr = new int[strArray.length];//int数组长度就是字符数组长度

 

              for (int x = 0; x < arr.length; x++) {

                     arr[x] = Integer.parseInt(strArray[x]);

              }

 

              // 对int数组排序

              Arrays.sort(arr);

 

              // 把排序后的int数组再组装成一个字符串

              StringBuilder sb = new StringBuilder();

              for (int x = 0; x < arr.length; x++) {

                     sb.append(arr[x]).append(" ");

              }

              //转化为字符串

              String result = sb.toString().trim();

             

              //输出字符串

              System.out.println("result:"+result);

       }

}
