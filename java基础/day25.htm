第二集    事件监听机制

 

         A:事件源   事件所要发生的地方

         C:事件    就是要发生的事情

举例：人受伤事件。

        

         事件源：人（具体的对象）

                   Person p1 = new Person("张三");

                   Person p2 = new Person("李四");

         事件：受伤

                   interface 受伤接口 {

                            一拳();

                            一脚();

                            一板砖();

                   }

         事件处理：

                   受伤处理类 implements 受伤接口 {

                            一拳(){ System.out.println("鼻子流血了，送到卫生间洗洗"); }

                            一脚(){ System.out.println("熏倒了，送到通风处"); }

                            一板砖(){ System.out.println("头破血流，送到医院"); }

                   }

                   p1.注册监听(受伤接口)

问题：

                当接口方法比较多时，实现类如果仅仅使用其中的一个方法，也得把其他的方                   法给实现，这样太麻烦。

解决方案：

                当接口方法比较多时，用适配器类实现接口,不过它仅仅是空实现，即方法体中不             提供内容，实现类再继承适配器类，用哪个方法就重写哪个方法。

 

适配器类的作用就是把含有多个方法的接口与单一需要的实现类结合起来，相当于         充电器/变压器的功能。

 

如果接口中只有一个方法，那么实现类直接实现这个接口，无需再用适配器类。好         比变压器输入为10伏，所需的输出也为10伏，还用变压器干啥？

 

 

//接口：

public interface UserDao {

         public abstract void add();

 

         public abstract void delete();

 

         public abstract void update();

 

         public abstract void find();

}

 

//适配器类：

public class UserAdapter implements UserDao {

 

         @Override

         public void add() {//空实现，方法体内无内容

         }

 

         @Override

         public void delete() {

         }

 

         @Override

         public void update() {

         }

 

         @Override

         public void find() {

         }

 

}

 

//实现类：

public class UserDaoImpl extends UserAdapter {//继承自适配器类，需要谁就重写谁

         @Override

         public void add() {

                   System.out.println("添加功能");

         }

}

所以以后在写事件监听器的时候，先写事件接口，如果发现接口中的方法只有一个，那么就这样用；如果发现接口中的方法很多，那么找到所需要的方法，再用适配器改进。

public class FrameDemo

{

         public static void main(String[] args)

                   Frame f = new Frame("添加按钮");

                            }

                   });

 

                   // 窗体显示

           B:如图：把文本框里面的数据转移到文本域

 

public class FrameDemo

{

         public static void main(String[] args)

         {

                   // 创建窗体对象

                   Frame f = new Frame("数据转移");

                   // 设置窗体属性和布局

                   f.setBounds(400, 200, 400, 300);

                   f.setLayout(new FlowLayout());

// 设置窗体关闭

                   f.addWindowListener(new WindowAdapter()

                   {

                            @Override

                            public void windowClosing(WindowEvent e)

                            {

                                     System.exit(0);

                            }

                   });

 

                   // 创建文本框

                   final TextField tf = new TextField(20);

                   // 表示文本框中可以放20个字符

 

                   // 创建按钮

                   Button bu = new Button("数据转移");

 

                   // 创建文本域

                   final TextArea ta = new TextArea(10, 40);// 10行40列

 

                   // 把组件添加到窗体，要有先后顺序，否则与要求不合

                   f.add(tf);

                   f.add(bu);

                   f.add(ta);

 

                   // 对按钮添加事件

                   bu.addActionListener(new ActionListener()

                   {

                            @Override

                            public void actionPerformed(ActionEvent e)

                            {

                                     // 获取文本框的值

                                     String tf_str = tf.getText().trim();

                                     // trim()方法表示去掉前后空格。

                                     // 这个地方是内部类，而tf是在main方法内定义的局部变量，

                                     // 局部内部类访问局部变量,局部变量必须为final型，所以在tf                                        //定义处为final

                                     // 同样下面的ta变量也是如此

 

                                     // 清空数据

                                     tf.setText("");

 

                                     // 设置给文本域

                                     // ta.setText(tf_str);

                                     // 调用此方法，后面添加的内容会把前面添加的内容替换掉，

                                     // 所以应用下面的追加方法

 

                                     // 追加和换行

                                     ta.append(tf_str + "\n");

 

                                     // 每次转移数据后，文本框会失去光标，

                                     // 为了转移数据后文本框能再得到光标,所以用此方法

                                     tf.requestFocus();

                            }

                   });

 

                   // 设置窗体显示

                   f.setVisible(true);

         }

}

                   f.setLayout(new FlowLayout());

                   f.addWindowListener(new WindowAdapter()

                   {

                            @Override

                            public void windowClosing(WindowEvent e)

                            {

                                     System.exit(0);

                            }

                   });

 

                   // 创建Label标签对象

                   Label label = new Label("只能输入数字：");

                   TextField tf = new TextField(40);// 文本框输入40个字符的长度

 

                   // 添加到窗体上

                   f.add(label);

                   f.add(tf);

 

                   // 给文本框添加键盘事件

                   tf.addKeyListener(new KeyAdapter()

                   {// KeyListener接口有三个方法，所以这里用适配器，重写键盘输入事件                       //的方法

                            @Override

                            public void keyPressed(KeyEvent e)

                            {

                                     // 如果输入的字符不是数字字符就取消事件

                                     // 思路：先获取字符，再判断字符，不是数字就取消事件

                                     // char getKeyChar()

                                     char ch = e.getKeyChar();

                                     System.out.println(ch);

                                     if (!(ch >= '0' && ch <= '9'))

                                     {

                                               e.consume();// 取消事件

                                     }

                            }

                   });

 

                   // 设置窗体可见

                  f.setVisible(true);

         }

}
