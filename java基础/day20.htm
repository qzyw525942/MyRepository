                C:遍历该File数组，得到每一个File对象

                                  否：不搭理它

 

public class FilePathDemo {

         public static void main(String[] args) {

                  // 封装目录

                  File srcFolder = new File("E:\\JavaSE");

 

                  // 递归功能实现

                  getAllJavaFilePaths(srcFolder);

         }

 

         private static void getAllJavaFilePaths(File srcFolder) {

                  // 获取该目录下所有的文件或者文件夹的File数组

                  File[] fileArray = srcFolder.listFiles();

 

                  // 遍历该File数组，得到每一个File对象

                  for (File file : fileArray) {

                           // 判断该File对象是否是文件夹

                           if (file.isDirectory()) {

                                    getAllJavaFilePaths(file);

                           } else {

                                    // 继续判断是否以.java结尾

                                    if (file.getName().endsWith(".java")) {

                                             // 就输出该文件的绝对路径

                                             System.out.println(file.getAbsolutePath());

                                    }

                           }

                  }

         }

}

public class FileDeleteDemo {

                  File srcFolder = new File("demo");

                  // 递归实现

                  deleteFolder(srcFolder);

         }

 

         private static void deleteFolder(File srcFolder) {

                  // 获取该目录下的所有文件或者文件夹的File数组

                  File[] fileArray = srcFolder.listFiles();

 

                           // 遍历该File数组，得到每一个File对象

                           for (File file : fileArray) {

                                    // 判断该File对象是否是文件夹

                                    if (file.isDirectory()) {

                                             deleteFolder(file);

                                    } else {

                                             System.out.println(file.getName() + "---" + file.delete());

//删除文件

                                    }

                       System.out.println(srcFolder.getName() + "---" + srcFolder.delete());

//删除文件所在的文件夹，因为此时它已经是空文件夹了

                  }

         }

}

第二集   IO流

 

      (1)用于在设备间进行数据传输的操作

      (2)分类：

                  A:流向(以程序为参照)

                           输入流     读入数据

                           输出流     写出数据

                  B:数据类型

                           字节流（byte）

                                             字节输入流 读取数据  顶层抽象类：InputStream

                                             字节输出流 写出数据  顶层抽象类：OutputStream

 

                           字符流（char）

                                             字符输入流  读取数据  顶层抽象类：Reader

                                             字符输出流  写出数据  顶层抽象类：Writer

                  注意：

                           a:字节流传输的是字节byte,字符流传输的是字符char。

                           b:如果操作的是电脑记事本打开能读懂的文本类型，才采用字符流，否则建议                    使用字节流。如果什么都不知道就用字节流。字节流可以处理字符流能处理的                       内容，不过没有字符流效率高而已

c:上面的四个顶层抽象类都不能直接实例化，所以要学习它们的实现类.

d:每个顶层抽象类的实现类都是以父类名作为后缀名：

            XxxOutputStream

                     XxxInputStream

                    XxxReader

                     XxxWriter
