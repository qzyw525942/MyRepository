第一集 常用类----StringBuffer类

  

1:StringBuffer

       (1)如果对字符串进行拼接操作，每次都要创建String对象，如String s1=”hello“；String s2=”world”;String s3=s1+2;  s1占用一个内存，s2占用一个内存，s3占用一个内存，比较耗时并且也耗内存，而这种拼接操作又是比较常见的，为了解决这个问题，Java就提供了一个字符串缓冲区类StringBuffer供我们使用.它是线程安全的可变字符串。

 

线程安全(在多线程部分具体讲解)

 安全即同步，指做一件事情依次执行，中间不能打岔，在此过程中其它程序不可访问此       程序的数据，这样可以保证数据是安全的

 不安全即不同步，指做一件事情中间有打岔，这样可以提高程序的执行效率·

 

  安全和效率问题是永远困扰我们的问题，二则是矛盾的。

 

  安全：医院的网站，银行网站（提高数据的安全性）

  效率：新闻网站，论坛之类的（提高程序的执行效率）

  

  StringBuffer和String的区别?

  前者长度和内容可变，后者不可变

  如果使用前者做字符串的拼接，不会浪费太多的资源。

              // 定义功能

              String s1 = arrayToString(arr);

              System.out.println("s1:" + s1);

              String s2 = arrayToString2(arr);

       // 用StringBuffer做拼接的方式

       public static String arrayToString2(int[] arr) {

              StringBuffer sb = new StringBuffer();

 

              sb.append("[");

              for (int x = 0; x < arr.length; x++) {

                     if (x == arr.length - 1) {

                            sb.append(arr[x]);

                     } else {

                            sb.append(arr[x]).append(", ");

                     }

              }

              sb.append("]");

 

              return sb.toString()//转换为String

       }

 

       // 用String做拼接的方式

       public static String arrayToString(int[] arr) {

              String s = "";

 

              s += "[";

              for (int x = 0; x < arr.length; x++) {

                     if (x == arr.length - 1) {

                            s += arr[x];

                     } else {

                            s += arr[x];

                            s += ", ";

                     }

              }

              s += "]";

 

              return s;

       }

}

         A:冒泡排序

                     相邻元素两两比较，大的往后放，第一次完毕后，最大值出现在了最大索引处。                     同理，再次从头开始两两比较，但最后一个元素无需再比较，因为第一次比较               中它已经是最大的元素了，所以只比较到长度-1的地方，所以把第二个较大                   的数字往后放在第二大的索引处，以此类推，其他的元素就可以排好。下面的                     原理图自己画一遍，能深刻理解原理。

       public static void bubbleSort(int[] arr)

       {

              for (int x = 0; x < arr.length - 1; x++)

              // 外循环控制次数，总的比较次数是数组长度减去1

              {

                     for (int y = 0; y < arr.length - 1 - x; y++)

                     // 每一次遍历的数组都要比上一次减去一个长度，如第一次比较后，

                     // 最大索引处已经是数组中的最大值了，第二次无需再比较它，所以

                     // 第二次比较的数组要减去1个长度，以此类推，

                     // 从第一次到第arr.length次（0~arr.length-1）,次数越来越大，

                     // 减去的也越来越多，即无须比较的越来越多，所以要减去x,

                     // 这一点仔细体会

                     {

                            if (arr[y] > arr[y + 1])//前后两个元素进行比较

                            {

                                   int temp = arr[y];

                                   arr[y] = arr[y + 1];

                                   arr[y + 1] = temp;

                            }

                     }

              }

       }

                     for(int y=x+1; y<arr.length; y++){//第一次比较是比较索引1及其以后的元素，第二次比较是比较索引2及其以后的元素，第三次比较是比较索引3及其以后的元素，所以每一次遍历的数组是从以当前次数为索引的元素及其以后的元素，又因为x表示当前次数-1，所以x+1表示当前次数

                            if(arr[y] <arr[x]){//当前元素和它后面的元素比较

                                   int temp = arr[x];

                                   arr[x] = arr[y];

                                   arr[y] = temp;

                            }

                     }

              }

       }

        B:二分查找(折半查找)：针对数组有序的情况

(千万不要对无序数组先排序再用二分查找,因为一旦排序就改变了原来的数组顺序)

                           

              A:定义最大索引，最小索引

            B:计算出中间索引

            C:拿中间索引的值和要查找的元素进行比较

                    相等：就返回当前的中间索引

                   不相等：

                           大   左边找

                           小   右边找

             D:重新计算出中间索引

                   大   左边找

                          max = mid - 1;因为已经不和中间值等，所以中间值要+1或者—1

                   小   右边找

                          min = mid + 1;

             E:回到B

public class ArrayDemo

{

       public static void main(String[] args)

       {

              // 定义一个数组

              int[] arr =

              { 11, 22, 33, 44, 55, 66, 77 };

 

              // 写功能实现

              int index = getIndex(arr, 33);

              System.out.println("index:" + index);

 

              // 假如这个元素不存在后有什么现象呢?

              index = getIndex(arr, 333);

              System.out.println("index:" + index);

       }

 

       /*

        * 两个明确： 返回值类型：int   参数列表：int[] arr,int value

        */

       public static int getIndex(int[] arr, int value)

       {

              // 定义最大索引，最小索引

              int max = arr.length - 1;

              int min = 0;

 

              // 计算出中间索引

              int mid = (max + min) / 2;//所有和数组相关的操作除以2效果都是一样的

 

              // 拿中间索引的值和要查找的值进行比较

              while (arr[mid] != value)

              {

                     if (arr[mid] > value)

                     {

                            max = mid - 1;

                     } else if (arr[mid] < value)

                     {

                            min = mid + 1;

                     }

 

                     // 加入判断，一旦min大于max，则不用再比较，肯定不存在了，如果不加                 此if语句，则会出现死循环

                     if (min > max)

                     {

                            return -1;

                     }

 

                     mid = (max + min) / 2;

              }

 

              return mid;//经过多伦循环比较之后，mid值一定是要查找的值，所以返回的是mid

       }

 

 

}

/*

 

统计一个字符串中大写字母字符，小写字母字符，数字字符出现的次数。(不考虑其他字符)

 

 *

 *         A:定义三个统计变量。

 *                int bigCont=0;

 *                int smalCount=0;

 *                int numberCount=0;

 *         B:键盘录入一个字符串。

 *         C:把字符串转换为字符数组。

 *         D:遍历字符数组获取到每一个字符

 *         E:判断该字符是

 *                大写      bigCount++;

 *                小写      smalCount++;

 *                数字      numberCount++;

 *         F:输出结果即可

 */

public class CharacterTest {

       public static void main(String[] args) {

              // 定义三个统计变量。

              int bigCount = 0;

              int smallCount = 0;

              int numberCount = 0;

 

              // 键盘录入一个字符串。

              Scanner sc = new Scanner(System.in);

              System.out.println("请输入一个字符串：");

              // 把字符串转换为字符数组。

              char[] chs = line.toCharArray();

                            bigCount++;

                     } else if (Character.isLowerCase(ch)) {

                            smallCount++;

                     } else if (Character.isDigit(ch)) {

                            numberCount++;

                     }

              }

 

              // 输出结果即可

              System.out.println("小写字母：" + smallCount + "个");
