第一集   运算符

 

对常量和变量进行操作的符号就叫做运算符

 

1、算术运算符

+  -  *  /  %  ++   --

 

两个int类型的数据相除，只能得到int类型的结果，如3/4=0，要想得到正确的值，只需要把两个数中的任何一个设置为浮点数即可：3*1.0/4=0.75

      

%是先作商再取余数的意思，3%4=3，3除以4，商为0，余数为3  

 

++,--运算符的使用：

              单独使用：

                     放在操作数的前面和后面效果一样。

            x++和++x是一样的

y--和--y是一样的

 

              参与运算使用：

                     放在操作数的前面，先自增或者自减，然后再参与运算（使用前）。

                     放在操作数的后面，先参与运算，再自增或者自减（使用后）。

                    

int c = ++a;

int d = --b;

                    

int c = a++;

int d = b--;

 

注意以下的计算：

第一题

int a = 10;

              int b = 10;

              int c = 10;

 

              a = b++; //a=10,b=11,c=10。此处只有a和b变，c不变，所以c为10

              c = --a; //a=9,b=11,c=9。此处只有a和c变，b不变，所以b为前一题的值11        b = ++a; //a=10,b=10,c=9

              a = c--; //a=9,b=10,c=8//后面的每一步的计算都是在前一步的基础上进行，而不            是在初始值上进行

 

 

第二题

int x = 4;

int y = (x++)+(++x)+(x*10);

 

蓝色括号里：括号里的结果为4，x变为5；

红色括号里：括号里的结果为6，x变为6；

绿色括号里：括号里的结果为60，x变为6；

 

千万记住：后面的每一步的计算都是在前一步的值的基础上进行，而不是在初始值上进行

 

+这个符号有三层意思：

1、加法：System.out.println(3+4);

2、正号：System.out.println(+4);//打印出4

3、字符串连接符：

System.out.println("hello"+'a'+1);//打印出helloa1

System.out.println('a'+1+"hello");//打印出98hello

 

2、赋值运算符

 

=：把右边的数据赋给左边

 

扩展的赋值运算符：+=,-=,*=,/=,%=

+= 把左边和右边做加法，然后赋值给左边。

 

int y=10;

y+=20;

System.out.println(y);//打印出30

 

 

面试题（这个面试题出现的概率有点高）：

              short s=1;

s = s+1;

      

              short s=1;

s+=1;

 

              上面两个代码有没有问题，如果有，哪里有问题。

 

第一个报错的问题是可能会损失精度，因为short型的s在参与运算时会自动转换

 为int型，所以s+1结果是int型的，而=左边的s是short型的，所以报错

但是第二个没有问题，因为扩展的赋值运算符其实隐含了一个强制类型转换。

s+=1;不是等价于s=s+1,而是等价于s=(short)(s+1),所以不会报错

4、逻辑运算符

 

&&  ||   !    和   &   |  ^

 

 

逻辑运算符一般用于连接boolean类型的表达式或者值

 

&逻辑与:有false则false，同真则真。

|逻辑或:有true则true，同假则假。

^逻辑异或:相同为false，不同为true。举例：情侣关系。男男,男女,女男,女女

!逻辑非:非false则true，非true则false。

 

 

&&和&的区别? 同理||和|的区别?

A:最终结果一样。

B:&&具有短路效果。当左边是false时，右边不执行。

             

开发中常用的逻辑运算符：

&&  ||  !

 

      

5、位运算符

 

位运算做的是二进制的补码的运算，主要存在于底层的源码中。所以要做位运算要把数据换为二进制且是补码。

 

&：位与运算

|：位或运算

^：位异或运算

~：按位取反运算

             

             

int a = 3;

int b = 4；

 

System.out.println(3 & 4);//当&两边跟的是boolean型的数据，做的是逻辑与，

                                                 //当跟的是数字，做的是位运算,|和^也一样

              System.out.println(3 | 4);

              System.out.println(3 ^ 4);

              System.out.println(~3);

 

分析：因为是位运算，所以我们必须先把数据换算成二进制的补码。

      

       3的二进制：11

              00000000 00000000 00000000 00000011

       4的二进制：100

              00000000 00000000 00000000 00000100

 

       又因为3和4都是正数，所以正数的原码，反码和补码都一样，应该用补码运算，       结果用原码表示

 

       &位与运算：有0则0（类似有假则假），同1则1。

              00000000 00000000 00000000 00000011（补码）

          &00000000 00000000 00000000 00000100（补码）

              -----------------------------------

              00000000 00000000 00000000 00000000（原码）

              结果是：0

             

       |位或运算：有1则1，同0则0。

              00000000 00000000 00000000 00000011（补码）

          |00000000 00000000 00000000 00000100（补码）

              -----------------------------------

              00000000 00000000 00000000 00000111（原码）

              结果是：7

             

       ^位异或运算：相同则0，不同则1。

              00000000 00000000 00000000 00000011（补码）

          ^00000000 00000000 00000000 00000100（补码）

              -----------------------------------

              00000000 00000000 00000000 00000111（原码）

              结果是：7

             

       ~按位取反运算符：0变1，1变0

              00000000 00000000 00000000 00000011（补码）

          ~11111111 11111111 11111111 11111100 (按位取反，这也是补码，要转化为原码)

         

          补码：11111111 11111111 11111111 11111100

          反码：11111111 11111111 11111111 11111011

          原码：10000000 00000000 00000000 00000100

          结果是：-4

 

^ 还有一个特点：某一个数据a被另一个数据b按位异或两次，结果还是这个数据a。

 

int a = 10;

int b = 20;

             

              System.out.println(a ^ b ^ b); //10

              System.out.println(a ^ b ^ a); //20

6、条件运算符（三目运算符）

 

              格式：比较表达式?表达式1:表达式2;

             

              比较表达式:结果是一个boolean类型。

             

              执行流程：

                     根据比较表达式的计算返回一个true或者false。

                     如果是true，就把表达式1作为结果。

                     如果是false，就把表达式2作为结果。

 

int x = 100;

              int y = 200;

              int z = ((x > y)? x: y);

System.out.println("z:"+z);//打印出200

 

练习：

//获取两个整数中的最大值

              int x = 100;

              int y = 200;

             

              int max = (x > y)? x: y;

              System.out.println("max:"+max);

 

      

                            //获取三个整数中的最大值

                            int a = 10;

                            int b = 30;

                            int c = 20;

             

                            //分两步：

                            //A:先比较a,b的最大值

                            //B:拿a,b的最大值再和c进行比较

                            int temp = ((a > b)? a: b);

                            int max1 = (temp > c? temp: c);

                            System.out.println("max1:"+max1);

两个练习题：

A:获取两个数据中较大的值

B:判断一个数据是奇数还是偶数,并输出是奇数还是偶数

 

import java.util.Scanner;

 

class IfTest {

       public static void main(String[] args) {

              //创建键盘录入对象

              Scanner sc = new Scanner(System.in);

             

              //获取两个数据中较大的值

              System.out.println("请输入第一个数据：");

              int a = sc.nextInt();

             

              System.out.println("请输入第二个数据：");

              int b = sc.nextInt();

             

              //定义一个变量接收最大值

              int max;

             

              if(a > b) {

                     max = a;

              }else {

                     max = b;

              }

             

              System.out.println("max:"+max);

              System.out.println("----------------");

             

              //判断一个数据是奇数还是偶数

              System.out.println("请输入你要判断的数据：");

              int x = sc.nextInt();

             

              if(x%2 == 0) {

                     System.out.println(x+"这个数据是偶数");

              }else {

                     System.out.println(x+"这个数据是奇数");

              }

       }

}

 

 

三目运算符和if语句之二的区别：

              三目运算符能实现的，都可以采用if语句实现。反之不成立。

             

              什么时候if语句实现不能用三元改进呢?

                     当if语句控制的操作是一个输出语句的时候就不能。

                     为什么呢?因为三目运算符是一个运算符，运算符操作完毕就应该有一个结果，                而不是一个输出。

 

上面两题的A题就能用条件运算符和if语句之二两者来实现，因为max可以作为三目运算符的结果

B题就只能用if语句之二来实现，因为System.out.println语句不能作为三目运算符的结果
