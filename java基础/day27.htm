第一集  反射

 

      (1)类的加载

 

要使用一个类，系统会先把它的class文件加载到内存，会并为它创建一个class文件对象。而所有类的class文件对象都是Class类的对象，所以实际上也就是创建了一个Class类的对象。

 

      (2)反射

                  通过Class对象，去使用class文件中的成员变量，构造方法，成员方法

 

Class类中包含了访问class文件中的成员变量，构造方法，成员方法的方法，但是 在Class类中成员变量被称为了Field，构造方法被称为了Constructor，成员方法被         成称了Method，它们三个都被定义成了类。

 

获取Class对象的方式：

 * A:通过Object类的getClass()方法

 * B:通过数据类型的静态属性class

 * C:通过Class类中的静态方法

 *            public static Class forName(String className)

 

开发中选择第三种方式，因为第三种是一个字符串，而不是一个具体的类名。这样  

就可以把这样的字符串配置到配置文件中去了。

public class Tool

{

         public void setProperty(Object obj, String propertyName, Object value) throws                    Exception

         {

                  // 根据对象获取字节码文件对象

                  field.setAccessible(true);

                  // 给对象的成员变量赋值为指定的值

                  field.set(obj, value);

         }

}

(2)模版设计模式：就是只定义一个算法的框架，而将具体的算法交给子类去实现。

 

// 需求：请计算出一段代码的运行时间

 

//框架类

public abstract class GetTime {

  public long getTime() {                                                               

                  long start = System.currentTimeMillis();

                 

                  code();

 

                  long end = System.currentTimeMillis();

 

                  return end - start;

         }

 

         public abstract void code();

}

 

//子类1：for循环

public class ForDemo extends GetTime {

 

         @Override

         public void code() {

                  for (int x = 0; x < 100000; x++) {

                           System.out.println(x);

                  }

         }

 

}

 

//子类2：复制视频

public class IODemo extends GetTime{

 

         @Override

         public void code() {

                  try {

                           BufferedInputStream bis = new BufferedInputStream(

                                             new FileInputStream("a.avi"));

                           BufferedOutputStream bos = new BufferedOutputStream(

                                             new FileOutputStream("b.avi"));

                           byte[] bys = new byte[1024];

                           int len = 0;

                           while ((len = bis.read(bys)) != -1) {

                                    bos.write(bys, 0, len);

                           }

                           bos.close();

                           bis.close();

                  } catch (IOException e) {

                           e.printStackTrace();

                  }

         }

        

}

A、枚举基础

   枚举是指将变量的值一一列出来,变量的值只限于列举出来的值的范围内。举例：一周只有7天，一年只有12个月等。枚举可以像新建类一样新建。

 

格式是：

public enum 枚举类名 {

    枚举项1，枚举项2，枚举项3…;

}

 

//基本定义格式：

public enum Direction {

   FRONT, BEHIND, LEFT, RIGHT;

}

 

//扩展版1

public enum Direction2 {

   FRONT("前"), BEHIND("后"), LEFT("左"), RIGHT("右");

 

   private String name;

 

   private Direction2(String name) {//给出了有参构造方法，故上面的四个枚举值必须给出值

      this.name = name;

   }

 

   public String getName() {

      return name;

   }

  }

}
