第一集  自定义属性

 

在昨天的SetActivity中，通过自定义控件中定义的setTitle等方法为控件设置属性。现在有新的需求：打开SetActivity对应的界面布局文件activity_setting，可以看到调用自定义组合控件的部分：

        <com.example.mobilesafetest.ui.SettingView

        android:id="@+id/setting_relativeLayout"

        android:layout_width="fill_parent"

        android:layout_height="wrap_content" >

</com.example.mobilesafetest.ui.SettingView>

 

现在要在上面这段代码中仿照TextView的属性android:text="xxx"一样设置setTitle()方法的功能，比如itheima:title=”xxx”，该如何实现？

 

可以参考一下系统是怎么定义android:text="xxx"的，然后仿照着操作即可。

打开D:\SDK\sdk\platforms\android-18\data\res\values\attrs.xml，查找TextView---->

 

可以见到TextView下面有bufferType，text，hint，textColor等属性。

 

所以可见系统是在values目录下的attrs.xml中定义控件的属性的，所以我们也可以这样做。

 

在项目的values中新建attrs.xml，内容如下：

<?xml version="1.0" encoding="utf-8"?>

<resources>

 

<declare-styleable name="com.itheima.mobliesafe75.ui.SettingView">

//自定义控件的全类名

 

        <attr name="title" format="string" />

//name:属性的名称,format:类型

 

        <attr name="des_on" format="string" />

        <attr name="des_off" format="string" />

 

//checkBox是手动进行设置的，所以这里没办法进行配置，就不管了。

    </declare-styleable>

</resources>

 

在自定义控件的调用的地方就可以如下：

<com.itheima.mobliesafe75.ui.SettingView

             android:id="@+id/sv_setting_update"

             android:layout_width="match_parent"

             android:layout_height="wrap_content"

             itheima:title="提示更新"

             itheima:des_on="打开提示更新"

             itheima:des_off="关闭提示更新"

             ></com.itheima.mobliesafe75.ui.SettingView>

 

并把所在的界面布局文件的名称空间仿照着如下修改：

系统自带：xmlns:android="http://schemas.android.com/apk/res/android"

自己定义：xmlns:itheima="http://schemas.android.com/apk/res/com.itheima.mobliesafe75"

(前面红色部分代表后面蓝色部分)

 

这样设置好后，就相当于设置了上面的itheima:title=”xxx”中的itheima。

 

如上配置后，还需要在自定义控件的代码SettingView中进行如下操作：

MD5Util工具类：

public class MD5Util

{

         // 银行卡 6位数字，将密码进行10-30次MD5加密：将上一次的结果再加密

 

         public static String passwordMD5(String password)

         {       

                   StringBuilder sb = new StringBuilder();

 

                   try

                   {

                            // 1.获取数据摘要器

                            // arg0 : 加密的方式:MD5加密

                            MessageDigest md = MessageDigest.getInstance("MD5");

 

                            // 2.将一个byte数组进行加密,返回的是一个加密过的byte数组,二进制的哈希                    //计算,md5加密的第一步

                            byte[] digest = md.digest(password.getBytes());

 

                            // 3.遍历byte数组

                            for (int i = 0; i < digest.length; i++)

                            {

                                     // 4.MD5加密

                                     // byte值 -128-127

                                     int result = digest[i] & 0xff;

                                     // MD5加密就是将byte值和int类型的255（即0xff）进行与运算，得到                               //一个int类型的正整数

 

                                     // 将得到int类型转化成16进制字符串

                                     // String hexString = Integer.toHexString(result)+1;

                                     // 不规则加密,又叫加盐。实际上现在MD5加密也是可以反向解密的了。

                                     // 一般的加密这儿不加1，如下面这句，为了不能实现反向解密，所以我                                      们在这儿加1

 

                                     String hexString = Integer.toHexString(result);

 

                                     if (hexString.length() < 2)

                                     {// 长度小于2，就在前面添加0

                                               sb.append("0");

                                     }

 

                                     sb.append(hexString);

                            }

 

                            return sb.toString;

 

                   } catch (NoSuchAlgorithmException e)

                   {

                            // 找不到加密方式的异常

                            e.printStackTrace();

                   }

 

return null;

         }

}

右下角的按钮使用的是状态选择器。状态选择器就是根据按钮不同的状态设置不同的图片给按钮。

在drawable目录下新建button.xml文件：

<?xml version="1.0" encoding="utf-8"?>

    <item android:state_pressed="true"//pressed :按下的时候显示图片

          android:drawable="@drawable/function_greenbutton_pressed" /> 

        

<item android:drawable="@drawable/function_greenbutton_normal" />

//default :默认显示图片

    //上面的这两张图片都是.9.png格式的图片。这种图片不会因为拉伸导致变形

</selector>

 

然后在布局文件中Buttton节点下，使用android:background=”@drawable/button”就能引入上述状态选择器，这样当点击或不点击时就会改变效果。

SetUpBaseActivity如下：

（这个地方十分重要，一定要能理解，后面的智慧北京项目还会遇到）

public abstract class SetUpBaseActivity extends Activity

{

         // 按钮点击事件的操作

         // 将每个界面中的上一步,下一步按钮点击事件方法,抽取到父类中

         public void pre(View v)

         {

                   pre_activity();

         }

 

         public void next(View v)

         {

                   next_activity();

         }

 

         //让子类实现这个抽象方法,根据自己的特性去实现相应的操作

         // 下一步的操作

         public abstract void next_activity();

 

         // 上一步的操作

         public abstract void pre_activity();

 

         // 在父类中直接对子类中的返回键进行统一的处理

         // 监听手机物理按钮的点击事件

         // keyCode :　物理按钮的标示

         // event : 按键的处理事件

         @Override

         public boolean onKeyDown(int keyCode, KeyEvent event)

         {

                   // 判断keycode是否是返回键的标示

                   if (keyCode == KeyEvent.KEYCODE_BACK)

                   {

                            // true:屏蔽按键的事件

                            // return true;

 

                            pre_activity();

 

                   }

                   return super.onKeyDown(keyCode, event);

         }

//在调用方法的时候，先去父类中寻找方法，如果父类没有再去子类调用方法

//所以如果在这个父类中已经定义好了，就会执行这个父类中的方法

}

         protected void onCreate(Bundle savedInstanceState) {

                   super.onCreate(savedInstanceState);

                   setContentView(R.layout.activity_setup3);

         }

 

         @Override

         public void next_activity() {

                   // 跳转到第四个界面

                   Intent intent = new Intent(this, SetUp4Activity.class);

                   startActivity(intent);

                   finish();

         }

 

         @Override

         public void pre_activity() {

                   // 跳转到第二个界面

                   Intent intent = new Intent(this, SetUp2Activity.class);

                   startActivity(intent);

                   finish();

         }

}

 

本界面的界面布局文件如下：

其中选择联系人也用状态选择器来实现：

<?xml version="1.0" encoding="utf-8"?>

<selector xmlns:android="http://schemas.android.com/apk/res/android">

    <item android:state_pressed="true"

          android:drawable="@drawable/btn_green_pressed" /> <!-- pressed :按下的时候显示图片-->

    <item android:drawable="@drawable/btn_green_normal" /> <!-- default :默认显示图片-->

</selector>

 

在选择联系人按钮中增加如下属性：

android:background="@drawable/selector_contact_button"
