第一集  简介

 

JNI：java native interface， java本地接口，是一套Java和C语言之间互相调用的接口的协议标准，相当于翻译器。

 

为什么要使用JNI?

1、java可以通过JNI调用C语言写的硬件驱动，从而控制硬件

2、java运行效率比C语言低得多，java可以通过JNI把耗时操作放在C语言中去执行

3、java的反编译文件容易看懂，而C语言的反编译文件很难看懂，java可以通过JNI把重要的业务逻辑放在C语言中去执行

4、C语言的历史比java的历史长很多，期间积累的大量的类库，java可以通过JNI去使用这些类库

现在把数据交换部分抽取为函数：

         int j=456;

现在再来看看java中的效果：

 

public class test

{

         public static void main(String[] args)

         {

                   int i = 123;

                   int j = 456;

                   change(i, j);

                   System.out.println(i + "," + j);

//可见在java中这种方式也是无法进行两个数的交换

         }

 

         public static void change(int i, int j)

         {

                   int temp = i;

                   i = j;

                   j = temp;

         }

 

}

 

为什么会出现这种情况？实际上C语言和java是一样的原理：

 

基本数据类型的变量在传递值给其他方法使用时，传递的是值而不是变量本身，无论其他方法怎样改变这个值，对变量本身无影响。或者说基本类型中形式参数的改变对实际参数没有影响。

 

引用数据类型的变量在传递值给其他方法使用时，传递的是就是这个数据本身，其他方法一旦改变这个值，这个变量本身就发生了改变。或者说引用类型中形式参数的改变直接影响实际参数

 

出现以上两种情况的原因是:基本数据类型没有把自己的地址传递过去,而仅仅是把值传递过去;而引用数据类型把自己的地址和值一起传递过去了。

 

c语言中指针代表的正是地址值。利用它就可以把基本数据类型的地址和值一并传递出去：

 

#include <stdio.h>

#include <stdlib.h>

 

main() {

 

         int i=123;

         int j=456;

         change(&i,&j);

         printf("i的值：%d \nj的值：%d \n",i,j);

//可见利用这种方式也能实现两个数的交换

 

}

 

change(int* i , int* j) {

         int temp=*i;

         *i=*j;

         *j=temp;

}

 

 

再来一例：

 

#include <stdio.h>

#include <stdlib.h>

 

main() {

 

int i=1;

int j=2;

function(&i,&j);

printf("i的值：%d \nj的值：%d \n",i,j);

}

 

function(int* i,int* j) {

 

*i *=2; //前一个*表示指针，后一个*表示乘法

}

第十一集  C语言中的栈内存

 

#include<stdio.h>                                                                                                                                                                                                                                       

#include<stdlib.h>

 

/**

 栈内存(int a)是系统统一分配统一回收的，如果其所占据的内存空间被回收了，那么其他程序可能再次使用此内存空间，导致回收前后的同一个栈变量(int a)的值不一样

 栈内存大小是固定的,内存地址是连续的.栈内存对内存的分配就叫静态内存分配

 */

 

int* getData()

{

         int array[] = { 1, 2, 3, 4, 5 }; //char类型的数组最后才要添加一个\0

 

         printf(" %#x \n", &array);

 

         return &array;

}

 

int* getData2()

{

         int array[] = { 5, 4, 3, 2, 1 };

 

         printf(" %#x \n", &array);

 

         return &array;

}

 

main()

{

         int* pointer = getData();

         //pointer是栈内存变量，现在接受的是getData()返回的结果

 

         getData2();

         //现在调用getData2()函数，导致上面的pointer的内存空间被回收。但是getData()和getData2()返回值的内存大小是一样的，

         //所以getData2()返回值会再次使用getData()返回值的内存空间，这样会使getData2()返回值付给pointer变量

         //这样就会出现一个问题，代码中没有把getData2()付给pointer变量，但是程序运行中却赋值了。

 

         printf("%d,%d,%d \n",*(pointer+0),*(pointer+1),*(pointer+2));

         //如果上面没有调用getData2();则这里就打印出getData()函数的结果。反之打印出getData2()的结果

 

         printf("%d,%d,%d \n",*(pointer+0),*(pointer+1),*(pointer+2));

         //无论有没有运行getData2()函数，对同一个栈变量再次打印都会出现更乱的情况

 

         printf(" %#x \n",pointer);

         //如果上面没有调用getData2();则这里就打印出getData()函数中数组的第一个元素的地址值。反之打印出getData2()中数组的第一个元素的地址值。

         //实际上因为无论有没有运行getData2()函数，用的都是同一块内存空间，所以这里的值总是一样的

 

}

 

 

第十二集  C语言中的堆内存

 

#include<stdio.h>

#include<stdlib.h>

 

/**

 java中new一个对象就会申请一块堆内存

 c中使用malloc函数申请堆内存： memory allocation：内存分配之义

 c的堆内存需要程序员手动申请(malloc)手动释放(free)

 

 申请一块堆内存就叫动态内存分配

 

 堆内存是不连续的，大小不固定，取决机器的状态

 

 */

 

main()

{

         //malloc函数：向系统申请size个字节的内存空间。申请到的是一块连续的内存空间

         //参数表示申请的内存大小

         //返回一个内存地址值void* ,其可以强制转换为任何类型的指针int* pointer或者

//double* pointer等等。

         int* pointer = malloc(sizeof(int)*5);

//5表示申请连续的五个，每个为int大小的内存空间

// pointer指向的是这五个堆内存空间的第一个位置，后面的位置依次加1即可得到

         *(pointer+0) = 1;

         *(pointer+1) = 2;

         *(pointer+2) = 3;

         *(pointer+3) = 4;

         *(pointer+4) = 5;

 

         //C语言中for循环的临时变量i要先声明再使用

         int i;

         for(i = 0;i<5;i++)

         {

                   printf("第 %d 个元素的值= %d \n",i,*(pointer+i));

         }

 

         free(pointer);   //释放指针变量pointer所指向的内存空间

 

         printf("第一个元素的值 %d \n",*(pointer+0));

 

}

 

 

第十三集  综合案例：学生学号管理系统

 

#include<stdio.h>

/**

 */

 

main()

{

         //学生总数：

         int count;

         scanf("%d",&count);

 

         //申请一块堆内存，并分别存入学号

         int* pointer = malloc(sizeof(int)*count);

         int i;

         for(i = 0;i<count;i++)

         {

                   printf("请输入第 %d 个学生的学号:",i+1);

                   //个数从第1个开始，而i从0开始，所以加1

 

                   scanf("%d", pointer+i);

                   //pointer本身就是所申请的堆内存地址的第一个位置，后面要填写哪个位置的值，就加上代表这个位置的i

                   //注意scanf的第二个参数所需要的就是地址值

         }

 

         //打印出上面输入的所有学号：

         for(i = 0;i<count;i++)

         {

                   printf("第 %d 个学生的学号是: %d \n",i+1,*(pointer+i));

                   //取出每个地址中所存入的数据，所以使用*(pointer+i)

         }

 

         //输入插班生人数：

         printf("请输入插班生的人数:");

         //声明一个变量increment用来保存插班生的人数

         int increment;

         //接受用户的输入

         scanf("%d",&increment);

 

         //重新申请一块足够大的内存：

         //如果 malloc申请到的内存后面还有足够的内存， realloc会在malloc申请的内存后追加自己所需要的内存

         //如果 malloc申请到的内存后面没有足够的内存，realloc会找到一块足够大的堆内存，先把 malloc申请到的内存中的值复制过来，然后再往后追加自己所需要的内存

         //第一个参数是malloc所返回的指针变量pointer；第二个参数是malloc和realloc总的所需要的内存大小

         pointer = realloc(pointer,sizeof(int)*(count+increment));

         for(i = count;i<count+increment;i++)

         {

                   printf("请输入第 %d 个学生的学号:",i+1);

 

                   scanf("%d", pointer+i);

         }

 

         //打印出插班生的学号：

         for(i = count;i<count+increment;i++)

         {

                   printf("第 %d 个学生的学号是: %d \n",i+1,*(pointer+i));

         }

}

 

第十四集  C语言中的结构体

 

#include<stdio.h>

#include<stdlib.h>

 

void study()

{

         printf("good good study!\n");

}

 

typedef struct Student

//typedef：用来为复杂的声明定义简单的别名

//struct：用来声明c的结构体。结构体类似java的类

{

         int age;  //8字节

         int score;  // 4字节

         char sex;   //1字节

 

         //声明函数指针。

         void (*studypointer)();

         //c结构体中不能定义函数!!!

         //如果结构体中非得有函数， 可以采用函数指针来进行：

         //函数指针的定义：

         //返回值 (*函数指针变量名字)(函数参数);

         //返回值和函数参数都要和所指向的函数的返回值以及函数参数一致。中间的函数指针变量名字可以任意

 

} stud;

 

main()

{

         stud stu =

         {        18,100,'f'};

         //为stu初始化。类似java中的对象初始化

 

         stu.studypointer = &study;

         //使函数指针指向具体的函数

 

         stu.studypointer();

         //调用函数指针指向的函数。此处与调用java中的方法一样

 

         struct Student* stuPointer = &stu;

         //声明结构体指针。

 

         printf(" *stuPointer.age = %d \n",(*stuPointer).age);

         //同样的，*stuPointer其实就是stu

 

         (*stuPointer).sex ='m';

         printf("stu.sex = %c \n",stu.sex);

 

         printf("stuPointer->age = %d \n",stuPointer->age);

         //stuPointer->age：前一个是指针变量，后一个是结构体中的变量。

         //这一句的意思就是通过一级指针变量去访问其所指向的结构体中的变量。

         //也就相当于stu.sex

         //->叫做间接引用运算符

 

         //直接调用

         //printf("stu.age = %hd \n",stu.age);

         //printf("stu.score = %d \n",stu.score);

         //printf("stu.sex = %c \n",stu.sex);

 

         // printf("结构体student占 %d 个字节 \n",sizeof(stu));

         //1、 结构体的大小大于等于结构体中每一变量占字节数的和

         //2、 结构体的大小是最大的那个变量所占字节数的整数倍

         //从结构体中已经标明的字节来看，这里应该打印出16，刚好满足这两个条件

}

 

第十五集  C语言中的联合体

 

#include<stdio.h>

#include<stdlib.h>

 

union test

//union声明联合体

//联合体的用法和结构体的用法差不多。

{

         int num; //4字节

         double d; //8字节

}

 

main()

{

         union test u1;

         u1.num = 123;

         u1.d = 123.456;

 

         printf("union占 %d 个字节 \n", sizeof(u1));

         //联合体占字节数取决于其中成员占内存空间最大的那一个

         //上面联合体中最大的为8，所以这里打印出的就为8

 

         printf("u1.num = %d \n", u1.num);

         //联合体中的所有变量共用同一块内存。上面先对num赋值，然后再对d赋值，d就会把原来num的值覆盖掉。

         //然后这里再对num读取值，实际上是按照int类型去读取d的低4位的值了。

         //因此联合体多用于内存不大的地方，比如51单片机上

 

}

 

第十六集  C语言中的枚举

 

#include<stdio.h>

#include<stdlib.h>

 

enum weekday

//enum：用以声明枚举

{

         MON = 9, TUE, WEND, THUR, FRI, SAT, SUN

         //如果第一个变量不指定值，默认为0，后面的变量依次为1，2，3...

         //如果第一个变量指定为9，后面的变量依次为10，11，12...

};

 

main()

{

         enum weekday day = MON;

         //用法与结构体，联合体类似

 

         printf("day= %d ",day);

         //打印出9

 

}
